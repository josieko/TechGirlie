Flutter



Flutter is an open source framework by Google for building beautiful, natively compiled, multi-platform applications from a single codebase.

It is most popular to use for mobile app development but you can also use it to build web applications. It uses a programming language called Dart, which is similar to Java and JavaScript meaning you wouldn‚Äôt need to learn different languages for each platform like iOS or Android for example.

  • Single codebase meaning that all of the code for the application is stored in a single repository vs multi-codebase development, where different parts of the application are developed using separate codebases.

More about Single codebase vs Multi codebase

Benefits of using Flutter:

  • Time saving - Flutter's hot reload feature allows developers to see changes to their code reflected in the app immediately, without having to rebuild and reinstall the app. This can save developers a lot of time and effort.
  • Native performance - Flutter apps are compiled to native machine code, which means they perform just as well as native iOS and Android apps.
  • Cross-platform development - meaning it can be used to build mobile apps for both iOS and Android using a single codebase. This can save developers time and money, as they only need to write and maintain a single codebase for both platforms.
  • Simple - it comes with a wide range of pre-built widgets that developers can use to create beautiful and engaging user interfaces.

Used to build apps like Google pay, BMW, Bytedance, and eBay.

How to Set Up Flutter

If you decide to go for the official tutorial, it is good to have your Mac prepared.


We are using mac setup, but you can visit here if you use a different OS.


Make sure you have Xcode installed and your computer is meeting the minimum requirements below:

  • Operating Systems: macOS, version 10.14 (Mojave) or later.
  • Disk Space: 2.8 GB (does not include disk space for IDE/tools).
  • Tools: Flutter uses git for installation and upgrade. We recommend installing Xcode, which includes git, but you can also install git separately.

If installing on Apple Silicon Mac, make sure to have Rosetta translation environment by running sudo softwareupdate --install-rosetta --agree-to-license in terminal:





Download the Flutter SDK (Software Development Kit):

  • Flutter SDK Download Link

Once downloaded the zip file, extract it in the desired location by cd-ing to the location of your choice:
