So letâ€™s start with APIs first. What are they?

API, which stands for "Application Programming Interface", is a term you'll frequently encounter in the tech world. Think of an API as a magic gateway that enables different software applications to communicate and share information with each other. Just like in your everyday life, where you connect with friends through conversations, APIs allow software programs to connect and exchange data seamlessly.

Imagine you're using a weather app on your smartphone. The app fetches up-to-the-minute weather data from a weather service, like magic! Well, that magic is made possible by an API. The weather service provides an API that allows the weather app to request data like temperature, humidity, and forecasts.

If you watched Sex and the City, you know very well who Samantha Jones is. (we love her)



Imagine you're an aspiring writer, but don't have direct contacts with publishers and not sure where to start. That's where Samantha, our trusty PR girl, comes into play. Samantha is well-connected and has a knack for networking, just like an API connects and communicates with different software systems.

In this scenario, Samantha serves as your API. You (API user) reach out to Samantha with your manuscript (API request). She takes your book and talks to various publishers on your behalf to find a match for your manuscript.

Once Samantha has spoken with publishers (API provider), she comes back to you with the information on publishers that are interested in your book (API response). Samantha acts as the intermediary, simplifying the complex process of getting your book published.

Just like an API makes it possible for different software systems to exchange information seamlessly, Samantha connects you, the writer, with the publishers, making your publishing journey smoother and more efficient.

How the API works in this story:

    Writer (API user) -> Samantha (API) -> Publishers (API provider)

What can it be used for

Now that you understand better what it is, let's see how can we use APIs. Because they are versatile, just like you, they can be used for many things like:

  • Fetching data from a server such as the weather forecast or latest news and store products
  • Sending data to a server like when you upload a photo to social media or submit a form on a website
  • Performing actions on a server like when you send an email or create a new user account
  • Building new applications and services like a weather app that uses an API to get the weather forecast info to display it in the app
